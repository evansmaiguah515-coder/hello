# Introduction to GitHub for Version Control

This repository was created as part of my Cloud Computing class assignment.  
It demonstrates how to use *Git* and *Git Bash* for basic version control operations, including creating a repository, adding files, committing changes, and pushing them to GitHub.

## Tools Used
- Git  
- Git Bash  
- Visual Studio Code  
- GitHub

## Description
I created this repository to learn how to manage and share code using Git and GitHub.  
The project includes a sample file showing that I successfully used Git commands for version control.
